# PM Experiment Design

Use this skill for experiment plans, A/B test design, hypothesis writing, metric selection, segmentation, guardrail metrics, and readout templates.

## Workflow

1. Clarify decision, hypothesis, target population, intervention, expected behavior change, and risk.
2. Define primary metric, secondary metrics, guardrails, and minimum practical effect.
3. Choose experiment type: A/B, staged rollout, prototype test, concierge test, fake door, or cohort comparison.
4. Define eligibility, exclusions, duration, stopping rules, and analysis plan.
5. State what decision will be made for win, neutral, or loss.

## Output

- 实验背景与假设
- 分组与样本范围
- 指标体系
- 风险与护栏
- 实验流程与时间线
- 结果判读规则
- 后续决策

## Guardrails

- Do not recommend an A/B test when sample size, risk, or implementation cost makes it inappropriate.
- Avoid optimizing only short-term conversion if user trust or quality may degrade.
- Call out instrumentation requirements before launch.
